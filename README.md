# To-Do List App
Simple web-based to-do list built with plain HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as complete
- Remove tasks

## Usage
1. Clone the repo: `https://github.com/darshanchaudharii/To-Do-List`
2. Open `index.html` in your browser.
